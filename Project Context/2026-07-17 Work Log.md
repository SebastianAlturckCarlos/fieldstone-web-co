# 2026-07-17 — Enterprise Pivot, Pricing & Client Demos

Full-day overhaul of the Fieldstone Web Co site with Claude. See [[Site Map & Routes]] for the current structure reference.

## The pivot
Repositioned the entire site away from "$70 local websites" to **custom operational software (CRM, POS, client portals) for established multi-crew trade businesses** (HVAC, plumbing, electrical). All revenue-bracket language ("7-to-8 figure") was later removed in favor of an authoritative enterprise tone.

New landing page funnel: **Hero → Marquee → Pain Points → Platform → Security → Plans → Process → Contact**.

- **Hero:** "Your crews scaled. Your systems didn't." — ops-console mockup visual (dispatch rows, KPI tiles), CTA "Book a Discovery Call"
- **Pain Points:** quotes that leak revenue · dispatch by whiteboard · POS that talks to no one
- **Platform:** 5-stage workflow rail (lead → payment) + 3 module cards (Portals / CRM Core / POS & Payments) + "one encrypted data layer" bar
- **Security:** bank-level encryption, secure data architecture, 99.9% SLA, strict two-sided data privacy
- **Process:** ops audit → architect & build → deploy, train & scale
- **Contact:** discovery-call form with trade + fleet-size dropdowns (FormSubmit → sebastianrhoton@gmail.com)

## Pricing (final state after several iterations)
Pure MRR, zero setup fees ("Zero Setup Fees. Full Implementation & Onboarding Included." badge):

| Tier | Name | Price |
|---|---|---|
| 1 | Operations Kickstart | $399 / month |
| 2 | Growth CRM Suite ⭐ most popular | $799 / month · up to 5 seats |
| 3 | Custom Enterprise Architecture | Custom monthly SLA |

- Tiers 1–2 subtext: billed monthly, 12-month initial commitment
- Tier 3 CTA: "Book an Architecture Scoping Call"
- Earlier iterations (setup fees, per-user pricing, build-your-own module selector) were replaced by this — the selector was removed entirely so the section ends cleanly at the 3 cards.
